1. Project Title
FinTrack Compliance Settle – Financial Compliance Monitoring and Audit Management System

2. Problem Statement
Objective
FinTrack Compliance Settle is a web-based financial compliance management platform that enables organizations to monitor financial transactions, verify transaction records, detect suspicious activities, review flagged transactions, manage approval workflows, and assign audit tasks efficiently.
The system focuses on providing a centralized dashboard for compliance monitoring while ensuring accurate transaction tracking, risk analysis, auditor management, and regulatory compliance across multiple departments and company branches.

3. Project Description
FinTrack Compliance Settle is a React.js application developed to simplify financial compliance monitoring and auditing processes.
The platform allows users to:
● Monitor company transactions
● Manage compliance records
● Verify transaction authenticity
● Review suspicious transactions
● Analyze budget risks
● Manage approval workflows
● Assign audit tasks
● Search transaction records quickly
The application provides an efficient compliance management experience while demonstrating modern frontend development practices using React.js.

5. Features Implemented
Dashboard
Displays a centralized overview showing:
● Total Transactions
● Flagged Transactions
● Compliance Status
● Department Summary
Provides quick visibility into overall financial compliance activities.

Transaction Table
Displays transaction records including:
● Transaction ID
● Department
● Branch
● Amount
● Compliance Status
● Risk Level
Allows auditors to monitor all financial records from a single interface.

Transaction Form
Allows users to:
● Add Transactions
● Edit Transactions
● Update Compliance Data
Provides efficient transaction management and record maintenance.

Review Queue
Displays flagged transactions waiting for compliance review.
Purpose:
● FIFO Processing
● Organized Review Workflow
● Priority Monitoring

Approval Rules Hub
Stores and manages approval policies.
Features:
● Department Rules
● Approval Levels
● Transfer Policies
● Compliance Guidelines
Provides centralized approval workflow management.

Auditor Task Assignor
Automatically assigns audit tasks to auditors.
Purpose:
● Balanced Work Distribution
● Faster Case Resolution
● Efficient Resource Allocation

Quickest Problem Finder
Uses graph traversal algorithms to locate compliance issues quickly.
Implemented Using:
● BFS Algorithm
● Dijkstra Algorithm
Benefits:
● Faster Investigation
● Efficient Error Detection
● Reduced Audit Time

Search Bar
Allows quick searching of:
● Transaction IDs
● Departments
● Branches
● Compliance Status
Improves transaction lookup efficiency.
